Login/Registration

Client Request

username
PassBuddy username
hashed_password: hash(master_password, salt)
Hash the master password before submitting to PassBuddy to prevent PassBuddy from getting the key to decrypt all stord passwords.
encrypted_salt: symmetricEncrypt(salt, master_password)
Encrypt the salt used to hash the master password using the master password. This prevents PassBuddy Server from running rainbow table attacks.
public_key
Generated on registration. Friends use this to encrypt passwords for sharing with you.
encr_private_key: symmetricEncrypt(private_key, master_password)
Generated on registration. You use this to decrypt passwords shared with you. It is encrypted with your master password to prevent PassBuddy from decrypting shared passwords. (You don't decrypt the password directly, you decrypt and share this private key with a proxy which handles login for you.)
